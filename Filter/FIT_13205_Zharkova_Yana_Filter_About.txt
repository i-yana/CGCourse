При нажатии кнопки "File"->"open" открывается окно с примерами .bmp картинок. Выбранная картинка отображается в первой области. 
Щелкните "Edit"->"Select". Появится пунктирное окно. Двигая его по картинке, выберите область для редактирования.
Выберите подходящий фильтр в меню "ru.nsu.g13205.zharkova.Filter", появится картинка в третей области с примененным фильтром.
Некоторые фильтры требуют ввода параметров.
Для Дизеринга это количество цветов в палитре (от 2 до 256)
Для фильтров Робертса, Собеля регулируется порог.
Для гамма коррекции изменяется гамма.
Для поворота - угол.
Для отмены операции нажмите "Cancel" в диалоговом окне. Изображение вернется к исходному до редактирования выбранным фильтром. 
Кнопка "Update" применяет к изображению фильтр. 
Так же наглядно показываются изменения с помощью ползунка.
Чтобы сохранить картинку выберите "File"->"Save"
